# **App Name**: PosePerfect AI Cam

## Core Features:

- Real-time Pose Guidance: Provides live feedback and ghost pose overlays to guide users into flattering poses using on-device human pose detection.
- Pose Match Scoring: Analyzes the user's pose in real-time and provides a score to indicate how well it matches the suggested pose.
- Automatic Capture: Automatically captures the photo when the user achieves a 'perfect' pose based on AI assessment.
- Offline Functionality: The app functions entirely on-device, ensuring user privacy and enabling use without an internet connection.
- Number of People Detection: Uses a tool to detect the number of people in the camera frame in order to make suitable pose suggestions
- Visual and Haptic Feedback: Offers both visual cues (ghost overlays, skeleton feedback) and haptic feedback (vibrations) to assist users in achieving the desired pose.

## Style Guidelines:

- Primary color: A vibrant azure (#4285F4) to convey confidence and trustworthiness, inspired by professional photography and clarity.
- Background color: A light-scheme off-white (#F7F9FC), creating a clean and unobtrusive backdrop.
- Accent color: A complementary violet (#8F69DE) for highlighting key interactive elements and feedback cues.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a clean and objective feel, suitable for both headlines and body text.
- Simple, modern icons that visually represent different poses and settings.
- A clean and intuitive interface, prioritizing the camera view with unobtrusive overlay elements for pose guidance and feedback.
- Smooth transitions and subtle animations to provide real-time feedback and guide the user through the posing process.
